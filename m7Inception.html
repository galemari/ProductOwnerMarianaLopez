Cronograma MVP ECOMOVE - Grupo 6

Todo pensado para el MVP
MES 1
Fundación Técnica
Semana 1-2: Configuración de arquitectura base y APIs de transporte público
Semana 3-4: Desarrollo de funcionalidades core (búsqueda de rutas, integración básica)
MES 2
Integración y Desarrollo
Semana 5-6: Integración con 2 operadores de micro-movilidad (bicicletas/scooters)
Semana 7-8: Desarrollo del sistema de pago unificado y calculadora de CO2
MES 3
Testing y Lanzamiento
Semana 9-10: Testing exhaustivo, ajustes de UX y optimización de performance
Semana 11-12: Lanzamiento piloto con 100 usuarios beta en ciudad seleccionada

🎯 Hitos Clave del MVP

Mes 1: Prototipo funcional con transporte público básico
Mes 2: Integración completa con 3 tipos de transporte
Mes 3: Aplicación lista para usuarios beta con métricas de impacto

🚀 Objetivo del MVP

Validar la hipótesis de que los usuarios adoptarán una plataforma unificada para movilidad sostenible, con métricas objetivo de 70% de retención semanal y 4.0+ rating de satisfacción.
